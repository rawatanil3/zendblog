Instruction:

1. Unpack database.zip to your site folder.
2. You have to see {your sitefolder}/database.
3. Go to database.
4. You will see folders application, docs, library, public, tests and file localhost.sql.
5. Go to library folder.
6. Load to this folder Zend library.
7. Go to application/configs/ and open file application.ini.
8. Change resource parameters (host, username, password) to your own values.
9. Import file localhost.sql to your mysql environment.
10. In browser go to page {your site homepage}/database/public.
11. To log in use values 'user' = 'admin', 'password' = 'password'.
